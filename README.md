# Aircraft Tracker

Aircraft Tracker is an interactive web application that displays sample aircraft data on a Leaflet map.

## Features

- Displays aircraft markers on an interactive map
- Shows aircraft heading and direction
- Displays altitude, speed, country, and location
- Includes a cleaner selected-aircraft details panel
- Allows users to refresh aircraft data without reloading the whole webpage
- Updates the aircraft count and last refresh time
